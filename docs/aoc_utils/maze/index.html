Implementations of 2D mazes.

Structs

HashMapMaze
A 2D maze where each location and value in the maze is stored in a HashMap. The idea here is that each key in the hashmap uniquely identifies a location in a 2D grid. Therefore, a natural choice for the generic type L is (usize, usize)
HashMapOccupiedMaze
Wraps a HashMapMaze with a current location L to enable a set of associated functions for mutably moving around in the maze.
